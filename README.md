# Installation
You need:
* Node
* pnpm

clone this repository
```bash
git clone https://github.com/wolmeister/bigorna
```
go to /bigorna and run:
```bash
pnpm install
```
API

go to /bigorna/api and run:
```bash
pnpm run gen
```
Start server:
```bash
pnpm run dev
```

WEB

go to /bigorna/web and start server:
```bash
pnpm run dev
